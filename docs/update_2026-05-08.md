## 1. 주요 변경 사항

### 고확신 구간 정밀 분석 추가 (`shared`)

`train_regression.py`에만 있던 고확신 30% 구간 분석을 `shared` 파이프라인에도 이식함.

- 테스트셋에서 `|예측 로그수익률|` 기준 상위 30% 샘플만 추출
- **Long(상승 예측) / Short(하락 예측)** 정확도를 분리해서 계산
- 결과는 메타데이터 JSON에 저장되고 콘솔에도 출력됨

**출력 예시**
```
Direction accuracy: 56.10%
  [고확신 상위 30%] 기준 문턱값(LogRet 절대값): 0.4505
  상승(Long) 확신 시 정확도: 59.26%  (n=135)
  하락(Short) 확신 시 정확도: 40.00%  (n=5)
```

### 뉴스 피처 (12개) — `REGRESSION_STYLE_NEWS_FEATURE_COLUMNS`

| 피처 | 설명 |
|---|---|
| `news_count_5d` | 최근 5일 뉴스 건수 합산 |

| `title_sentiment_5d_mean` | 제목 감성 점수 5일 이동평균 |
| `negative_news_spike_5d` | 본문 부정 확률 / 최근 5일 평균 부정 확률 |
| `body_sentiment_decay_3d` | `body_sentiment_score × 0.5^(days_since_news / 3)` |

## 3. 학습 설정 (공통)

| 항목 | 값 |
|---|---|
| 예측 대상 | QQQ |
| 데이터 기간 | 2017-01-12 ~ 2026-05-02 |
| 고정 horizon | 5거래일 |
| Train / Test | 80% / 20% |
| Optuna trials | 200 |
| Random seed | 42 |
| 학습 타깃 | 5일 뒤 로그수익률 × 100 |
| Optuna 목적함수 | `direction_accuracy - rmse × 0.1` |
